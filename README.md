# Seismology-CH.3
# 地震學講義：第三章 - 地震學與地球結構 (Seismology and Earth Structure)

本目錄包含了地震學課程中關於「地震學與地球內部結構」的系列講義檔案。這份講義涵蓋了從基本的地震波原理、折射與反射地震學，到地球內部的非均向性、衰減特性以及地函與地核的組成成分等核心地物觀念。

## 講義目錄與大綱

以下是各小節講義的詳細說明：

* **[3.1 簡介 (Introduction)](./15_3.1%20Introduction.pdf)**
  * **檔案名稱**：`15_3.1 Introduction.pdf`
  * **內容摘要**：介紹地震學在探測地球內部震波速度分佈上的主要應用，為後續章節建立基礎概念。

* **[3.2 折射地震學 (Refraction seismology)](./16_3.2%20Refraction%20seismology.pdf)**
  * **檔案名稱**：`16_3.2 Refraction seismology.pdf`
  * **內容摘要**：探討平層模型 (Flat layer method) 的震波走時曲線，以及如何透過折射震波反演計算地下的地層速度結構。

* **[3.3 反射地震學 (Reflection seismology)](./17_3.3%20Reflection%20seismology.pdf)**
  * **檔案名稱**：`17_3.3 Reflection seismology.pdf`
  * **內容摘要**：討論反射震波在地球物理探勘上的應用，介紹如何利用反射訊號轉換為深度剖面，以了解地下地質構造。

* **[3.4 球體地球中的地震波 (Seismic waves in a spherical earth)](./18_3.4%20Seismic%20waves%20in%20a%20spherical%20earth.pdf)**
  * **檔案名稱**：`18_3.4 Seismic waves in a spherical earth.pdf`
  * **內容摘要**：將探討範圍從平層模型擴展至全球尺度，討論地震波在球體地球模型中的傳播與路徑幾何特性。

* **[3.5 體波走時研究 (Body wave travel time studies)](./19_3.5%20Body%20wave%20travel%20time%20studies.pdf)**
  * **檔案名稱**：`19_3.5 Body wave travel time studies.pdf`
  * **內容摘要**：介紹如何利用體波走時資料 (Travel time data) 搭配赫格洛茲-維歇特方法 (Herglotz-Wiechert method) 來決定地球內部隨深度的震波速度分佈。

* **[3.6 非均向性地球結構 (Anisotropic earth structure)](./20_3.6%20Anisotropic%20earth%20structure.pdf)**
  * **檔案名稱**：`20_3.6 Anisotropic earth structure.pdf`
  * **內容摘要**：跳脫理想的等向性假設，討論地球介質的非均向性 (Anisotropy)，探討地震波在非均向性介質 (如：上地函、內核) 中傳播時的特徵及虎克定律擴展。

* **[3.7 衰減與非彈性 (Attenuation and anelasticity)](./21_3.7%20Attenuation%20and%20anelasticity.pdf)**
  * **檔案名稱**：`21_3.7 Attenuation and anelasticity.pdf`
  * **內容摘要**：探討地震波能量在地球內部傳遞時的衰減機制 (品質因子 Q 值) 以及地球內部真實材料的非彈性特性。

* **[3.8 地函與地核的組成 (Composition of the mantle and the core)](./22_3.8%20Composition%20of%20the%20mantle%20and%20the%20core.pdf)**
  * **檔案名稱**：`22_3.8 Composition of the mantle and the core.pdf`
  * **內容摘要**：結合上述的地震波速、衰減等觀測數據，進一步推論並介紹地函與內外地核的礦物與化學組成。
